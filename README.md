# Netlify Deployment

This is a static resume website for Pratham Khanna.

## Files

- `index.html`: The main HTML structure displaying the resume.
- `styles.css`: CSS styles with dark/light mode support.
- `script.js`: JavaScript for theme toggling.

## Usage

1. Open `index.html` in a web browser to view the resume.
2. Click the "Toggle Dark/Light Mode" button to switch themes.
3. The theme preference is saved in local storage.

## Deployment

This is a static site, suitable for deployment on Netlify, GitHub Pages, or any static hosting service. Simply upload the files or push to a repository.
